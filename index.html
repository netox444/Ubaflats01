Regras de Convivência

Para manter um ambiente agradável para todos, pedimos atenção às regras abaixo:

  • ✓
    Mantenha o silêncio após 22h para respeitar o descanso dos demais hóspedes.
  • ✓
    Desligue luzes, ar-condicionado e aparelhos eletrônicos ao sair do flat.
  • ✓
    Preserve móveis, colchões, utensílios e estrutura do espaço.
  • ✓
    Feche portas e janelas ao deixar o ambiente.
  • ✓
    Realize o descarte do lixo em local apropriado.
  • ✓
    Não é permitido fumar dentro dos ambientes fechados.
  • ✓
    Visitas externas devem ser previamente autorizadas.
  • ✓
    Utilize os equipamentos do flat com cuidado.
  • ✓
    Evite consumir alimentos sobre camas e sofás para preservar a limpeza.
  • ✓
    Não mova móveis pesados sem necessidade.
  • ✓
    Em caso de problemas ou necessidade, fale conosco pelo WhatsApp.
